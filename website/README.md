# Bazardrom — маркетплейс автозапчастей (Бишкек)

Прототип сайта по брифам и аналитике конкурентов (Drom, Avito, Lalafo).

## Запуск

Откройте `index.html` в браузере или:

```bash
cd website
python -m http.server 8080
```

Откройте http://localhost:8080

## Страницы

- `index.html` — главная, поиск, УТП, мини-гайд
- `parts.html` — каталог запчастей с фильтрами
- `masters.html` — автоэлектрики
- `booking.html` — запись к мастеру
- `cabinet.html` — личный кабинет (история, записи)
- `help.html` — помощь (3 раздела)

## Особенности из брифа

- Дизайн в духе Drom (оранжевый/белый), без перегруза
- Обязательные цены в сомах (без «договорной»)
- Публикация только через WhatsApp + модерация
- Возврат запчасти 3 дня
- Запись к мастеру по дате/времени
- Диагностика от 500 сом

Замените номер WhatsApp в `js/data.js` (`WHATSAPP`).
